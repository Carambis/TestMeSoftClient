Вы находитесь на главной странице обучающего веб-ресурса тестирования программного обеспечения.

В настоящее время тестирование – процесс анализа программного средства и сопутствующей документации с целью выявления дефектов – стало обязательной частью жизненного цикла по разработке программного обеспечения. Тестирование направлено на обнаружение и устранение как можно большего числа дефектов. Следствием такой деятельности является повышение качества программного обеспечения по всем его характеристикам.

Знание основных принципов и умение тестировать программные продукты – одно из базовых требований, которое предъявляется к современному специалисту, работающему в сфере разработки программного обеспечения.

Данный веб-ресурс предоставляет следующие возможности:

1. Изучение теоретических данных с помощью методического пособия ЮЗАБИЛИТИ-ТЕСТИРОВАНИЕ ПРОГРАММНОГО ОБЕСПЕЧЕНИЯ (Авторы: М.М. Меженная, Т.В. Гордейчук, М.М. Борисик, О.С. Медведев, И.Ф. Киринович)

2. Решение практических задач, в которых главной целью является нахождение заключенных дефектов пользовательского интерфейса и функциональности (всего 20 заданий).

3. Предоставление персональных рекомендаций на основе полученного результата тестирования. Для дальнейшей работы рекомендуется перейти к изучению теоретической части.
